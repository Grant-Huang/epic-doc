## 快速入口

- **想快速看到效果**：运行 Streamlit 示例并在页面里直接生成 DOCX/HTML/PDF
  - 见：[界面使用说明（Streamlit）](ui.md)
- **想批量生成**：使用 `epic-doc generate` 从 JSON 生成 DOCX（可选同时导出 HTML/PDF）
  - 见：[CLI 使用说明](cli.md)
- **想在你的项目里集成**：直接 import `EpicDoc`/`build_from_dict`
  - 见：[API 使用说明](api.md)
