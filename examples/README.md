# examples

这里是 epic-doc 的示例入口。

## 快速运行示例脚本

在项目根目录：

```bash
python examples/basic_report.py
python examples/with_charts.py
python examples/complex_table.py
python examples/with_flowchart.py
```

## 帮助文档入口

### 1) 直接阅读 Markdown（GitHub 可渲染）

- `docs/index.md`
- `docs/design.md`
- `docs/ui.md`
- `docs/cli.md`
- `docs/api.md`

### 2) 在 Web 示例中查看（可渲染显示）

```bash
pip install streamlit
streamlit run web/streamlit_app.py
```

打开页面后切换到“帮助文档”页签即可阅读 `docs/*.md`。
